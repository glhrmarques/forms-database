**Simple Form Integration with Supabase**

This project focuses on the end-to-end process of designing, developing, and deploying a data-capture form integrated with a backend database.

Stack: Design (Figma), fronend(HTML/JS/CSS), backend (Node.js/Express), Database (Supabase).
